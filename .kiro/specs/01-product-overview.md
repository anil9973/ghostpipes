# GhostPipes - Product Overview

## Vision

Yahoo Pipes reborn as a modern, AI-assisted data automation platform that works as both a browser extension and web app. Empowers non-technical users while remaining powerful enough for data scientists.

## Core Philosophy

- **AI as Plumber**: AI builds pipelines, users supervise
- **Offline-First**: Extension works without internet, syncs when available
- **Visual Debugging**: See data flow in real-time through realistic pipes
- **Progressive Complexity**: Simple for beginners, powerful for experts

## Key Differentiators vs Competitors

### vs Azure Logic Apps

- **Simplicity**: No learning curve, AI generates everything
- **Visual**: Realistic pipes, not abstract flowcharts
- **Browser-Native**: No server needed for basic use

### vs Zapier

- **Transparency**: See and edit every transformation step
- **Data-First**: Work with actual data from start, not abstract triggers
- **Ownership**: Runs in your browser, your data stays local

### vs Make.com

- **AI-Driven**: Describe intent, don't assemble blocks
- **Contextual**: Extension understands what you're looking at
- **Free Tier**: Full features in extension mode (offline)

## Success Metrics (Phase 1 - Extension)

- Users can build their first pipeline in < 2 minutes
- 80% of pipelines generated by AI require zero edits
- Average pipeline completion time: < 30 seconds
- Error recovery rate: 90% (pipelines continue despite single node failures)

## User Journey (Extension - Phase 1)

### Discovery

1. User visits product page on Amazon
2. Thinks: "I want to track this price"
3. Installs GhostPipes extension

### First Use

1. Presses `Alt+E` on product page
2. Selects product name + price
3. Presses `Alt+P`
4. Floating menu suggests: **"Price Tracker"** template
5. Clicks template
6. Pipeline builder opens with data pre-filled
7. AI suggests: "Check daily at 9 AM?"
8. User accepts
9. Pipeline saved, first run scheduled

## Technical Constraints (MVP)

### Must Have

- Works offline (IndexedDB storage)
- No login required for basic features
- Chrome 140+ only (File System Access API)
- Max pipeline: 20 nodes (prevent complexity)
- Max execution: 5 minutes (prevent infinite loops)

### Nice to Have (Post-Hackathon)

- Firefox support (polyfill File System API)
- Safari support (limited features)
- Mobile companion app
- Pipeline analytics dashboard

## Spooky Theme Rationale

**Why ghosts/pipes metaphor works:**

- Pipes = familiar (Yahoo Pipes nostalgia)
- Ghosts = data invisibly flowing through system
- Dark theme = professional, not childish
- Subtle horror = memorable, not gimmicky

**Brand Personality:**

- Professional yet playful
- Powerful yet approachable
- Technical yet artistic
